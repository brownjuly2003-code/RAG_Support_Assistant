## Proof points

<CardGrid>
  <Card title="Published docs hub" icon="document">
    Docs are published under `/RAG_Support_Assistant`. English and Russian
    entry pages are maintained explicitly.
  </Card>
  <Card title="Seeded support KB" icon="open-book">
    The demo KB starts from three seed documents: warranty, returns, and
    E10-E30 errors. E20 is present in `errors_e10_e30.md`.
  </Card>
  <Card title="Fixture-backed evaluation" icon="check-circle">
    The fixture set has 12 cases across error codes, password reset, warranty,
    installation, billing, and general support. The three-document demo KB does
    not cover every fixture category.
  </Card>
  <Card title="Traceable integration surface" icon="puzzle">
    The API catalog is tied to FastAPI route decorators, and `/api/ask`
    returns an answer with documents and citations.
  </Card>
</CardGrid>
