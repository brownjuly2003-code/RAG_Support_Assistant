## Доказательства

<CardGrid>
  <Card title="Опубликованный docs hub" icon="document">
    Документация опубликована под `/RAG_Support_Assistant`. Английская и
    русская стартовые страницы поддерживаются явно.
  </Card>
  <Card title="Засеянная демо-база" icon="open-book">
    Демо-база начинается с трех документов: гарантия, возвраты и ошибки
    E10-E30. E20 есть в `errors_e10_e30.md`.
  </Card>
  <Card title="Фикстура для оценки" icon="check-circle">
    Фикстура содержит 12 кейсов: ошибки, сброс пароля, гарантия, установка,
    биллинг и общие вопросы. Демо-KB из трех документов покрывает не все
    категории фикстуры.
  </Card>
  <Card title="Проверяемая API-поверхность" icon="puzzle">
    Каталог API связан с FastAPI route decorators, а `/api/ask` возвращает
    ответ вместе с документами и citations.
  </Card>
</CardGrid>
